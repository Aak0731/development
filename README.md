# 长线养成游戏升级经验曲线设计
## 这个项目是做什么的

为1-90级的长线养成游戏设计升级经验数值。
核心思路是用幂函数控制成长节奏，并在60-80级区间设置资源“软卡点”，让玩家前期升级快、中期有追求感、后期平稳养成。

## 设计公式
经验值 = 50 × 等级^1.3 + 100，结果四舍五入取整。
50到60级的时候开始上调参数，使得60-80的经验变化不会太夸张。
60到80级的经验需求在公式基础上额外上浮30%，形成软卡点。
由于游戏设计中升到80级即可进行进阶或者突破之类的操作，故80-90级这段可以降低参数，避免玩家想要升到90的迫切的心被阻拦

## 项目文件
curve_design.xlsx 是1-90级的经验配表。
calculate.py 是模拟脚本，模拟零氪、月卡、双月卡三种玩家每天升级的过程，输出升级曲线图和每日升级记录。

## 运行方法
安装pandas、matplotlib、openpyxl三个库，然后运行calculate.py即可。
